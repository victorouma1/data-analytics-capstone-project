# Global Cost of Living and Salary Analyser
### Problem statement
In an increasingly globalised world individuals are more open to relocating for better financial opportunities. However higher salaries do not always translate into better living standards or savings due to variations in cost of living ,rent and local economic conditions. There is a need for a data driven solution that helps individuals compare cities and countries based on real purchasing power and potential savings rather than salary alone.
### Desription
This project aims to build an end-to-end data analytics solution that analyzes and compares cost of living and salary data across global cities.

### Research Questions
Which cities offer the highest absolute savings, (salary minus cost of living)? <br>
Which cities offer the best savings ratio (relative affordability)? <br>
How does cost of living vary across regions and countries? <br>
Are there geographic patterns in high-salary vs high-cost cities? <br>
How much of income is typically spent on rent vs other expenses? <br>
Which factors contribute most to high living costs? <br>
How do cities within the same country differ? <br>
Which countries provide the best balance between income and expenses? <br>
